# Quick Start Guide

Get up and running in 5 minutes!

## 1. Install Dependencies

```bash
# Backend
cd backend
npm install

# Frontend (in a new terminal)
cd frontend
npm install
```

## 2. Start Servers

```bash
# Terminal 1: Start backend
cd backend
npm start

# Terminal 2: Start frontend
cd frontend
npm run dev
```

## 3. Open Browser

Go to: http://localhost:5173

## 4. Create First Question

1. Go to http://localhost:5173/admin
2. Login with password: `admin123`
3. Click "Start Game" to activate
4. Click "+ New Question"
5. Copy/paste from `EXAMPLE_QUESTIONS.json`
6. Click "Create"

## 5. Play!

1. Go to http://localhost:5173
2. Click "Get Started"
3. Enter your name
4. Answer questions
5. See your score!
